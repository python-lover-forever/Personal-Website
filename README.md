# Ahmed Farouk Portfolio

presents my personal portfolio website which showcases my work as a full-stack engineer and my passion for robotics and my role as an entrepreneur who founded both **Adapt Non-profit** and **E3lan 3almashy**.

The website was developed using **HTML** and **CSS** and **JavaScript**.

Live demo:  

https://python-lover-forever.github.io/Personal-Website/index.html

## Features

- The website displays a dark cyber aesthetic which uses cyan accents for its design
- The website features a custom robot cursor that tracks the mouse movement
- The title animation on the hero section displays a glitching effect
- The story uses a typing terminal effect to present its content
- The background particle dust field creates an interactive experience *reset the field by pressing the R key
- The website design works perfectly on all devices
- The website includes a top position scroll progress bar
- Users can play the maze mini-game through a separate tab which opens the game in `maze.html`
- The contact section provides actual links which enable users to reach out through email and phone and LinkedIn

## Technologies Used

- HTML5
- CSS3 (with modern animations & mix-blend-mode)
- Vanilla JavaScript
- The Canvas API enables users to create particles and develop the maze game
- AudioContext API (scroll beep sound)
